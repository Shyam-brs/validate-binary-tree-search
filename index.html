Validate Binary Search Tree (BST)

isValidBST function: This function takes a root node and optional minimum and maximum values as input, and returns a boolean indicating if the tree is a valid BST. If the root node is null, the function returns true. If the root node's value is less than or equal to the minimum value or greater than or equal to the maximum value, the function returns false. Otherwise, the function calls itself recursively with the left and right child nodes, passing the current node's value as the maximum value for the left child and the minimum value for the right child.
